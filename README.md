# Landing Page Khơi - GitHub Pages

Upload toàn bộ file trong thư mục này lên repository `khoi` của tài khoản `thucphamhaigia`.

Sau khi upload:
1. Vào Settings → Pages
2. Source: Deploy from a branch
3. Branch: main / root
4. Save

URL dự kiến: https://thucphamhaigia.github.io/khoi/

Các mục cần cập nhật sau:
- Điện thoại/Zalo
- Fanpage
- Ảnh chai thật hoặc ảnh chụp sản phẩm chuyên nghiệp
- Giấy chứng nhận muối nếu muốn công khai
